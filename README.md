# AtPartyIntro

A simple Amiga 500 intro made for presentating at @Party2025.

To build and run you'll need to be on Windows using VSCode with Bartman's C/C++ plugin installed.

You can find Bartman's plugin in the marketplace or here:
https://github.com/BartmanAbyss/vscode-amiga-debug
