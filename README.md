# AtPartyIntro

A simple Amiga 500 intro made for presentating at @Party2025.

To build and run you'll need to be using VSCode with Bartman's C/C++ plugin installed.
It's setup for Windows but should work on Mac/Linux apart from the data build batch file and bitmap conversion tools, which aren't necessary.

You can find Bartman's plugin in the marketplace or here:
https://github.com/BartmanAbyss/vscode-amiga-debug
